# 进阶教程

本章节包含 RMTDataPro 的进阶使用教程。

## 处理流程详解

典型的 RMT 数据处理流程包括以下环节：

```mermaid
    graph TB
        subgraph 数据加载
            A[SBF 数据] --> B[频段解析]
            B --> C[D1-D4 频段提取]
        end
        
        subgraph 阻抗估计
            C --> D[目标频点选择]
            D --> E[Gamble 方法估计]
            E --> F[张量/标量阻抗]
        end
        
        subgraph 后处理
            F --> G[质量评估]
            G --> H[曲线绘制]
            H --> I[数据导出]
        end
```

## 学习路径

建议按以下顺序学习：

1. 掌握基本操作流程（快速入门）
2. 了解 SBF 数据格式结构
3. 理解 RMT 与 MT 的处理差异
4. 实践批量导出功能
5. 探索 Z 曲线叠加分析
