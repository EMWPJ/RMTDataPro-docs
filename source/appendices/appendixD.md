# 附录 D：常见问题

本文档解答 RMTDataPro 使用中的常见问题。

## 📋 通用问题

### Q1: 软件启动失败怎么办？

**A:** 检查以下可能原因：

1. **缺少 Visual C++ 运行库**
   - 下载并安装 [Visual Studio 2022 Redistributable](https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist)

2. **Qt 插件加载失败**
   - 确认 Qt 环境变量正确设置
   - 使用 `-Deploy` 参数重新部署 Qt 依赖

3. **显卡驱动过旧**
   - 更新显卡驱动到最新版本

### Q2: 工程文件无法打开？

**A:** 可能原因及解决方法：

| 原因 | 解决方法 |
|------|----------|
| 文件损坏 | 从备份恢复 |
| 版本不兼容 | 更新软件到最新版本 |
| 路径包含中文/特殊字符 | 移动到纯ASCII路径 |

### Q3: 处理速度很慢？

**A:** 优化建议：

1. **增加窗口重叠率** - 减少处理窗口数
2. **减小窗口长度** - 降低频率分辨率
3. **关闭详细日志** - 减少IO开销
4. **使用 SSD** - 提升数据读写速度

## 📂 数据问题

### Q4: SBF 文件加载失败？

**A:** 检查以下内容：

1. **文件格式**: 确认是 SBF 格式（非 AGE/SEG-D）
2. **文件完整性**: 尝试重新下载或复制文件
3. **版本兼容**: 部分旧版 SBF 可能不兼容

### Q5: 某些频段数据缺失？

**A:** 可能原因：

- 该频段在采集时未启用
- 采集设备不支持该频段
- 数据损坏导致解析失败

### Q6: 处理结果异常（曲线抖动）？

**A:** 数据质量问题：

1. **信噪比低**: 延长采集时间
2. **人文干扰**: 远离电力线、建筑物
3. **设备问题**: 检查校准状态

## ⚙️ 参数问题

### Q7: FFT 参数如何选择？

**A:** 参考建议：

| 应用场景 | 窗口长度 | 重叠率 | 窗口模式 |
|----------|----------|--------|----------|
| 快速浏览 | 256 | 0.5 | 单窗口 |
| 标准处理 | 512 | 0.5-0.75 | 多窗口 |
| 高精度 | 1024 | 0.75 | 多窗口 |

### Q8: 张量阻抗和标量阻抗如何选择？

**A:** 选择建议：

| 类型 | 适用场景 |
|------|----------|
| 张量阻抗 | 三维结构、极化分析 |
| 标量阻抗 | 一维/二维结构、噪声较大数据 |

## 📤 导出问题

### Q9: EDI 导出格式不被识别？

**A:** 检查 EDI 文件格式：

- 确保包含 `>HEAD` 和 `>MT DATA` 标记
- 确保 `>END` 标记存在
- 确认频率值为正数

### Q10: 批量导出失败？

**A:** 可能原因：

1. **输出目录不存在或无写入权限**
   - 创建目录或更换输出路径
2. **文件名冲突**
   - 启用"覆盖"选项
3. **磁盘空间不足**
   - 清理磁盘空间

## 🌐 语言问题

### Q11: 界面语言无法切换？

**A:** 解决方法：

1. 确认翻译文件存在：
   - `RMTDataPro_zh_CN.qm`
   - `RMTDataPro_en_US.qm`
2. 重启软件使语言切换生效
3. 检查系统区域设置
