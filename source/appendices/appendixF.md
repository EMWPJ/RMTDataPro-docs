# 附录 F：术语表

本附录定义了 RMTDataPro 和 RMT 数据处理中使用的专业术语。

## 📡 RMT 术语

### 基础概念

| 术语 | 英文 | 说明 |
|------|------|------|
| **射频大地电磁法** | Radio Magnetotellurics (RMT) | 使用高频电磁信号的地电磁探测方法 |
| **大地电磁法** | Magnetotellurics (MT) | 使用天然电磁场的地球物理方法 |
| **视电阻率** | Apparent Resistivity (ρₐ) | 表征地下电性分布的等效电阻率 |
| **相位** | Phase (φ) | 阻抗的相位角，反映电性界面的相位响应 |

### 频段术语

| 术语 | 说明 |
|------|------|
| **D1** | 低频段，采样率 39 kHz |
| **D2** | 中频段，采样率 312 kHz |
| **D3** | 高频段，采样率 832 kHz |
| **D4** | 极高频段，采样率 2496 kHz (M模式) 或 1248 kHz (L模式) |

## 📊 数据处理术语

### FFT 相关

| 术语 | 英文 | 说明 |
|------|------|------|
| **窗口长度** | Window Length | FFT 分析的样本点数 |
| **重叠率** | Overlap | 窗口之间的重叠比例 |
| **窗函数** | Window Function | 用于减少频谱泄漏的加权函数 |
| **频率分辨率** | Frequency Resolution | 相邻频率点的间隔 |
| **多窗口谱分析** | Multiple Taper Spectral Method (MTSM) | 使用多个正交窗函数的谱分析方法 |

### 阻抗估计

| 术语 | 英文 | 说明 |
|------|------|------|
| **张量阻抗** | Tensor Impedance | 完整的 2×2 阻抗张量 |
| **标量阻抗** | Scalar Impedance | 简化的单值阻抗表示 |
| **传输函数** | Transfer Function | 输入输出信号的比值 |
| **互功率谱** | Cross Power Spectrum | 两个信号在频域的互相关 |

## 📂 文件格式术语

| 术语 | 说明 |
|------|------|
| **SBF** | Station Binary Format，测站二进制格式 |
| **EDI** | Electrical Data Interchange，MT 数据交换格式 |
| **CSV** | Comma Separated Values，逗号分隔值 |
| **JSON** | JavaScript Object Notation，JSON 数据格式 |

## 🔧 软件界面术语

| 术语 | 说明 |
|------|------|
| **测线** | Line，同一测线上的测点集合 |
| **测点** | Site，独立的数据采集站 |
| **工程** | Project，包含多条测线的完整数据集 |
| **批处理** | Batch Process，一次处理多个测点 |

## 📈 质量评估术语

| 术语 | 英文 | 说明 |
|------|------|------|
| **相干度** | Coherence | 信号质量指标，0-1 之间 |
| **标准误差** | Standard Error | 多次测量的离散程度 |
| **相对误差** | Relative Error | 相对于测量值的百分比误差 |

## ⚡ 电磁学术语

| 术语 | 符号 | 说明 |
|------|------|------|
| 磁场 | **H** | 磁场强度矢量 |
| 电场 | **E** | 电场强度矢量 |
| 磁感应强度 | **B** | B = μ₀H |
| 阻抗 | **Z** | E/H 的比值 |
| 角频率 | **ω** | 2πf |
