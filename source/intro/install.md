# 使用指南

RMTDataPro 是免安装软件，无需安装，直接运行即可使用。

## ⚙️ 系统要求

### 最低系统要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10/11（64位） |
| 处理器 | Intel/AMD x64 处理器 |
| 内存 | 8GB RAM |
| 存储 | 2GB 可用磁盘空间 |
| 显卡 | 支持 OpenGL 3.3 |

### 推荐系统配置

| 项目 | 推荐配置 |
|------|----------|
| 操作系统 | Windows 11（64位） |
| 处理器 | Intel Core i5 / AMD Ryzen 5 及以上 |
| 内存 | 16GB RAM 及以上 |
| 存储 | SSD，5GB 可用空间 |

## 🚀 快速开始

1. **获取软件**
   - 从发布页面下载最新版本压缩包
   - 解压到任意目录

2. **运行程序**
   - 双击 `RMTDataPro.exe` 启动软件
   - 无需安装，绿色运行

## 📁 目录结构

```
RMTDataPro/
├── RMTDataPro.exe    # 主程序
├── Qt/                # Qt 运行时库
├── Intel/             # Intel MKL 库
└── ...
```

## ❓ 常见问题

### Q: 提示缺少 DLL 文件

确保 `Qt` 和 `Intel` 目录与 `RMTDataPro.exe` 在同一目录下。

### Q: 无法启动

- 检查系统是否满足最低要求
- 确保显卡驱动已更新
- 尝试以管理员身份运行
