9. 数据导入

本章介绍如何将外业采集的 RMT 数据导入 RMTDataPro 工程，包括 SBF 格式和 TR 格式数据的导入方法。

9.1. 📂 数据格式简介

RMTDataPro 支持两种常用的数据格式：

9.1.1. SBF 格式（频谱格式）

SBF（Station Binary Format）是 RMT 野外采集的频谱数据格式，包含预先计算的频谱信息。

特点：

  • 二进制格式，文件体积小

  • 已包含 FFT 频谱数据

  • 支持多频段（D1/D2/D3/D4）

支持的频段：

9.1.2. TR 格式（时间序列格式）

TR 格式是原始的时间序列数据格式，包含未处理的电磁场时间序列。

特点：

  • 文本或二进制格式

  • 需要软件进行 FFT 处理

  • 保留完整原始数据信息

9.2. 📥 数据导入操作

9.2.1. SBF 文件导入

通过菜单导入

  1. 选择 项目 → 新建工程（或打开已有工程）

  2. 在工程管理面板中，选中目标测线

  3. 右键点击 → 导入 SBF 文件

  4. 在文件对话框中选择要导入的 SBF 文件

  5. 点击”打开”确认导入

通过拖拽导入

  1. 打开工程并选中目标测线

  2. 直接将 SBF 文件拖拽到工程管理面板

  3. 软件自动识别并导入文件

9.2.2. TR 文件导入

TR 格式文件的导入流程与 SBF 类似：

  1. 选择 项目 → 打开工程

  2. 选中目标测线

  3. 右键 → 导入 TR 文件

  4. 选择 TR 格式文件并确认

注意：TR 格式导入后，需要在 FFT 处理阶段进行频谱计算。

9.3. 🗂️ 工程管理

9.3.1. 创建测线

  1. 在工程管理面板中右键点击工程节点

  2. 选择 新建测线

  3. 输入测线名称

  4. 测线创建完成

9.3.2. 创建测点

  1. 右键点击测线节点

  2. 选择 新建测点

  3. 输入测点名称

  4. 测点创建完成，可以导入数据
