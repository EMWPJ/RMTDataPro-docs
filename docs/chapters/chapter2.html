6. 数据导入与频段处理

本章介绍 RMTDataPro 中如何导入 SBF 数据文件、选择处理频段，以及进行阻抗估计的基本操作。

6.1. 📂 SBF 数据导入

6.1.1. 导入方式

RMTDataPro 提供两种数据导入方式：

方式一：通过菜单导入

  1. 选择 项目 → 打开工程（或新建工程）

  2. 在工程管理面板中，选中目标测线或测点

  3. 右键点击 → 导入 SBF 文件

  4. 在文件对话框中选择要导入的 SBF 文件

  5. 点击”打开”确认导入

方式二：通过拖拽导入

  1. 打开工程并选中目标测线

  2. 直接将 SBF 文件拖拽到工程管理面板

  3. 软件自动识别并导入文件

6.1.2. 支持的频段

SBF 数据包含四个频段，不同频段适用于不同的探测深度：

6.1.3. 导入后检查

导入完成后，在测点下会显示对应的频段数据：

  • ✅ D1 - 39kHz

  • ✅ D2 - 312kHz

  • ✅ D3 - 832kHz

  • ✅ D4 - 2496kHz

如果某个频段显示为 ❌ 或缺失，说明该 SBF 文件不包含该频段数据。

6.2. 📡 频段选择与处理

6.2.1. 选择目标频段

在数据处理窗口中：

  1. 展开测点节点，显示所有可用频段

  2. 勾选要处理的频段（可多选）

  3. 常用做法是先处理 D3 或 D4 频段进行浅层探测

6.3. 🔄 阻抗估计模式

RMTDataPro 支持两种阻抗估计模式：

6.3.1. 标量阻抗（Scalar Impedance）

  • 适用场景: 探测区域电性结构较为均匀、二维特征不明显

  • 优点: 计算简单、抗干扰能力较强

  • 配置: 在参数配置中选择”标量阻抗”模式

6.3.2. 张量阻抗（Tensor Impedance）

  • 适用场景: 探测区域存在明显二维/三维电性特征

  • 优点: 可揭示地电结构的走向特征

  • 配置: 在参数配置中选择”张量阻抗”模式

6.3.3. 选择建议

6.4.2. 检查步骤

  1. 处理完成后，查看结果显示窗口

  2. 观察 ρ-φ 曲线的平滑程度

  3. 标记相干度较低的测点

  4. 必要时重新采集或更换测点位置
