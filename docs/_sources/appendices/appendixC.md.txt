# 附录 C：参数参考

本文档列出 RMTDataPro 中所有可配置的参数及其说明。

## ⚙️ FFT 参数

### 基础参数

| 参数名 | 类型 | 默认值 | 取值范围 | 说明 |
|--------|------|--------|----------|------|
| `windowLength` | int | 512 | 128-4096 | FFT 窗口长度（点数） |
| `overlap` | double | 0.5 | 0.0-0.95 | 窗口重叠率 |
| `windowMode` | enum | Multiple | Single/Multiple | 窗口模式 |
| `singleWindowType` | enum | Hanning | Hanning/Hamming/Kaiser | 单窗口窗函数 |
| `timeBandwidthProduct` | double | 2.0 | 1.0-4.0 | 多窗口时间带宽积(NW) |
| `numWindows` | int | 3 | 2-8 | 多窗口数量(K) |
| `bandwidth` | int | 3 | 1-7 | 频率提取带宽 |

### 阻抗参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `impedanceType` | enum | Tensor | Tensor（张量）/Scalar（标量） |
| `averageType` | enum | WeightedMean | Mean（均值）/WeightedMean（加权均值） |
| `dataType` | enum | SBF | SBF/TR 数据类型 |

### 频段启用

| 数据类型 | 频段 | 采样率 | 默认启用 |
|----------|------|--------|----------|
| SBF | D1 | 39 kHz | ✓ |
| SBF | D2 | 312 kHz | ✓ |
| SBF | D3 | 832 kHz | ✓ |
| SBF | D4 | 2496 kHz | ✓ |
| TR | T1 | - | ✓ |
| TR | T2 | - | ✓ |
| TR | T3 | - | ✓ |

## 📊 处理结果参数

### 阻抗张量

| 分量 | 符号 | 单位 | 说明 |
|------|------|------|------|
| Zxx | Zxx | Ω | 张量阻抗 xx 分量 |
| Zxy | Zxy | Ω | 张量阻抗 xy 分量 |
| Zyx | Zyx | Ω | 张量阻抗 yx 分量 |
| Zyy | Zyy | Ω | 张量阻抗 yy 分量 |

### 视电阻率与相位

| 物理量 | 符号 | 单位 | 计算公式 |
|--------|------|------|----------|
| 视电阻率 | ρ | Ω·m | ρ = \|Z\|² / μ₀ω |
| 相位 | φ | ° | φ = arg(Z) |

## 🎨 图表参数

| 参数 | 选项 | 默认值 |
|------|------|--------|

## 🔧 系统参数

### 内存限制

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 单次处理最大数据量 | 100 MB | 防止内存溢出 |
| 缓存大小 | 50 MB | 校准缓存 |

### 并行处理

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 最大线程数 | CPU核心数 | 自动检测 |
| 进度更新间隔 | 100 ms | UI更新频率 |
