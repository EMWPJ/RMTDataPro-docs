# 附录 A：键盘快捷键

本文档列出 RMTDataPro 中所有可用的键盘快捷键。

## 📋 项目操作

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl+Shift+N` | 新建工程 | 创建新项目 |
| `Ctrl+Shift+O` | 打开工程 | 打开已有工程 |
| `Ctrl+Shift+R` | 最近工程 | 打开最近项目列表 |
| `Ctrl+Shift+S` | 保存工程 | 保存当前工程 |
| `Ctrl+Alt+Shift+S` | 另存为 | 工程另存为 |
| `Ctrl+Shift+W` | 关闭工程 | 关闭当前工程 |
| `Ctrl+Shift+E` | 批量导出 | 批量导出 ρ-φ 曲线数据 |
| `Ctrl+Alt+Shift+X` | 退出 | 退出程序 |

## 🔧 设置快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl+Alt+Shift+F` | FFT 参数 | 打开 FFT 参数设置 |
| `Ctrl+Alt+Shift+C` | 校准 | 打开校准设置 |
| `Ctrl+Alt+T` | 样式设置 | 打开样式设置 |
| `Ctrl+Alt+S` | 图表系列设置 | 打开图表系列设置 |

## 📖 帮助

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `F1` | 关于 | 打开关于对话框 |

## 🖱️ 鼠标快捷键

| 操作 | 功能 | 说明 |
|------|------|------|
| **双击测点** | 打开详情 | 打开测点详情窗口 |
| **右键菜单** | 上下文菜单 | 显示右键操作菜单 |
| **Ctrl+滚轮** | 缩放图表 | 放大/缩小曲线视图 |
| **拖拽** | 移动视图 | 平移图表视图 |

## 🔤 自定义快捷键

目前版本暂不支持自定义快捷键配置。
