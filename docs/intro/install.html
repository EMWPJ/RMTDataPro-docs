2.3. 🔧 构建步骤

2.3.1. 1. 环境准备

确保已安装以下软件：

  1. Visual Studio 2022 (17.0+)

    • 下载地址: Visual Studio Downloads

    • 安装时选择 “使用 C++ 的桌面开发”

  2. Qt 6.9+

    • 下载地址: Qt Downloads

    • 安装组件: Qt 6.9.1 MSVC 2022 64-bit

  3. Intel oneAPI MKL 2024.2+

    • 下载地址: Intel oneAPI

    • 组件: Intel oneAPI Base Toolkit

2.3.2. 2. 获取源码

# 克隆仓库
git clone https://github.com/EMWPJ/EMDataPro.git

# 进入项目目录
cd EMDataPro

2.3.3. 3. 使用 PowerShell 脚本构建

项目提供了便捷的构建脚本：

# 配置项目（使用 MSVC，Release 模式）
.\config.ps1

# 使用 Intel 编译器配置
.\config.ps1 -Compiler intel

# 使用 Ninja 生成器
.\config.ps1 -Compiler msvc -Generator ninja

# 启用 OpenMP
.\config.ps1 -UseOpenMP

# 清理并重新配置
.\config.ps1 -Clean

2.3.4. 4. 编译项目

# 基本编译
.\build.ps1

# Debug 模式
.\build.ps1 -BuildType Debug

# 编译特定目标
.\build.ps1 -Target RMTDataPro

# 指定并行任务数
.\build.ps1 -Parallel 8

# 部署 Qt 依赖到输出目录
.\build.ps1 -Deploy

2.3.5. 5. 运行程序

编译完成后，可执行文件位于：

build/msvc_release/bin/RMTDataPro/RMTDataPro.exe

运行程序：

& "D:\code\EMDataPro\build\msvc_release\bin\RMTDataPro\RMTDataPro.exe"

2.4. 📁 目录结构

构建后的目录结构：

build/msvc_release/
├── bin/
│   ├── RMTDataPro/          # RMTDataPro 可执行文件
│   │   └── RMTDataPro.exe
│   └── tests/               # 测试程序
├── lib/                     # 库文件
└── qmake/                   # Qt 项目文件

2.5. ❓ 常见问题

2.5.1. Q: 编译时报错 “Qt not found”

确保设置了 Qt 环境变量或使用 -CMAKE_PREFIX_PATH 指定 Qt 安装路径：

.\config.ps1 -QtRoot "C:/Qt/6.9.1/msvc2022_64"

2.5.2. Q: Intel MKL 链接错误

确保 Intel oneAPI 环境变量已正确设置：

& "C:/Program Files (x86)/Intel/oneAPI/setvars.bat"

2.5.3. Q: 缺少 vcruntime.dll

安装 Visual Studio 2022 Redistributable： Visual Studio Redistributable
